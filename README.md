# How to use
- [download for mac](https://github.com/Hermanya/listenable-browser/releases/latest)
- run (unarchive, then right click and open)
- **cmd+b** to unlock/lock browser
- then you can:
    * `open messenger`
    * `open sound cloud`
    * `open youtube.com`
- when in doubt type `help`
